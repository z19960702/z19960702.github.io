hexo-command

  • blog

整理 Hexo 中一些常用的命令

最常用命令

hexo new “post title with whitespace” 新建一篇文章
hexo new page –path about/me “About me” 新建一篇文章（设置路径和标题）
hexo publish 发表草稿
hexo generate（hexo g） 生成静态文件 -d 文件生成后立即部署网站
hexo server(hexo s) 启动服务器 -p 重设端口
hexo deploy（hexo d） 部署网站 -g 部署之前预先生成静态文件

其他命令（偶尔会用到）

hexo init blog 在blog文件夹中新建一个网站
hexo clean 清除缓存文件 (db.json) 和已生成的静态文件 (public)。在某些情况（尤其是更换主题后），如果发现您对站点的更改无论如何也不生效，您可能需要运行该命令。
hexo list 列出网站资料
hexo version 显示 Hexo 版本

附加命令（还没用到过）

hexo render 渲染文件
hexo migrate 从其他博客系统 迁移内容
hexo –safe 在安全模式下，不会载入插件和脚本
hexo –debug 在终端中显示调试信息并记录到 debug.log
hexo –silent 隐藏终端信息
hexo –draft 显示 source/_drafts 文件夹中的草稿文章
hexo –cwd /path/to/cwd 自定义当前工作目录（Current working directory）的路径
