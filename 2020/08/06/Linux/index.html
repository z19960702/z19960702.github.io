Linux

对Linux进行学习

shell是唯一通过命令行接口（CLI）或图形用户接口（GUI）解释用户命令的用户接口

Linux为命令添加参数通常可以用以下两种方式之一：一个破折号后跟一个字母，或者两个破折号来引用该参数更为冗长的形式（如ls–human-readable会得出与ls-h完全相同的输出）

文件系统

ls 当前目录下，可以列出文件及子目录的名字
参数： -l 列出文件的权限、所有者、组文件大小以及时间戳等信息
-h 以用户可读的格式——KB（千字节）、MB（兆字节）以及GB（千兆字节）来显示文件的大小
-R 显示子目录及它们包含的文件与子目录
pwd 显示当前路径
cd 通过相对路径或绝对路径访问对应的文件夹
file 查看文件类型

touch 快速地创建一个空文件
mkdir 快速地创建一个空文件夹
rm 删除一个文件 -r 递归删除 -f 强制删除
rmdir 删除一个文件夹
cp 将对象复制至指定目录（可以识别复制位置是否为文件夹，若没有则进行创建）
mv 将对象移动至指定目录 要移动或复制多个文件，并且想要避免逐个输入所有的文件名，通常你可以使用通配符（*）来进行全局操作（正则表达）

命令使用

type 说明如何解释命令名
which 显示会执行哪些可执行程序
man 查看命令对应的文档 按下/键可以切换至屏幕底部的文字输入区，在这里可以输入要搜索的内容
apropos 显示一系列合适的命令
info 查看命令的info条目
whatis 查看一条命令的简述
通过使用分号来分隔多条命令，就可以将多条命令输入在一行中。command1;command2;command3……

I/O输出

I/O重定向功能可以重新定义标准输出位置。使用重定向操作符>，后面接文件名，就可以把标准输出重定向到另一个文件中，而不是显示在屏幕上，重定向符>>将使得输出内容添加在文件的尾部，重定向符2>将标准错误进行重定向，重定向符&>把标准输出和标准错误都进行重定向
| 可以把一个命令的标准输出传送到另一个命令的标准输入中
cat 将文件内容打印到输出
less 显示文件内容，可以上下翻页进行查看 G跳到文本文件尾 g跳到文件头 /进行查找 n查找下一个
sort 对文本行排序
uniq 报告或删除文件中重复的行
wc 打印文件中的换行符、字和字节的个数
grep 打印匹配行（过滤器）
head 输出文件的第一部分内容
tail 输出文件的最后一部分内容
tee 读取标准输入的数据，并将其内容输出到标准输出和文件中

journalctl 访问所有的系统日志
