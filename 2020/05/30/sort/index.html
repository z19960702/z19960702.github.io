十大排序算法

1.冒泡排序

冒泡排序就是依次两两相比，如果前一个数较大，进行交换，每一轮循环总能找到最大值，最大值就和水中的泡泡一样慢慢的从水底浮出水面。

改进：设置标志位，每一轮都对数组进行扫描，若数组已经变为有序则停止排序。

2.选择排序

选择排序就是依次在所有未排序的数中找出最小值放到队伍前端。

3.插入排序

插入排序就是从队首开始依次排序，将新检索的数放入已排好序中的相应位置。
def insectionSort(nums: List[int]):
length = len(nums)
for i in range(length):
preIndex = i-1
current = nums[i]
while preIndex >= 0 and nums[preIndex] > current:
nums[preIndex+1] = nums[preIndex]
preIndex-=1
nums[preIndex+1] = current
return nums

4.希尔排序

希尔排序就是先跳着排序，当步宽慢慢缩小时，队列也就越来越有序。

5.快速排序

快速排序就是选取一个数做标准，将比它小的放前边，比它大的放后边，然后再对选取数的左右两个区域开始递归（套娃），递归结束也就排序完成。属于分治思想

减少空间的使用

6.归并排序

归并排序就是先分，分到最后每一部分只剩一个数之后，再两两对比开始合并。同属于分治思想

7.堆排序

堆排序就是按照大根堆（或小根堆），每次取出根节点的数，对剩下的二叉树结构再进行调整形成大根堆，循环往复。

8.计数排序

计数排序就是典型的牺牲空间提升时间，使用哈希表进行存储，再依次取出即可完成排序。

9.桶排序

桶排序就是画出不同范围，先将数依次归到相应的范围中，再在范围里进行排序。

10.桶排序

基数排序就是依次按每个数的位先排好，当所有位都排好序，整个序列也就排好了。
